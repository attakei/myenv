# attakei.zsh

zsh, souce config, and zplugin for my workstation.

## Requirements

Need enabled to using git module.

## Role Variables

- `zsh_zplugin_dir`: Directory for zplugin. (default is `$HOME/.zplugin`) 

## Example Playbook

```
- hosts: servers
  roles:
    - attakei.zsh
```
